# Sistema Bancário em Python

Projeto simples de sistema bancário feito em Python no terminal.

## Funcionalidades

* Ver saldo
* Depositar valores
* Sacar valores
* Validação de saldo insuficiente
* Tratamento de erros (entrada inválida)
* Execução contínua até o usuário sair

## Exemplo de uso

=== SISTEMA BANCÁRIO ===
1 - Ver saldo
2 - Depositar
3 - Sacar
4 - Sair

Escolha: 2
Valor para depositar: 100

Saldo atual: R$ 100.00

## Como executar

```bash
python sistema_bancario.py
```

## Tecnologias

* Python

## Objetivo

Projeto criado para praticar lógica de programação, estruturas condicionais, tratamento de erros e simulação de regras de negócio.
